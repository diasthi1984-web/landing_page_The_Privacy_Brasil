# The Privacy Brasil - Landing Page

This project is a React-based landing page for **The Privacy Brasil**, featuring a custom scroll-driven animation and a modern corporate design.

## Prerequisites

- **Node.js**: You need to have Node.js installed to run this project. Download it from [nodejs.org](https://nodejs.org/).

## Setup & Running

1.  **Install Dependencies**:
    Open your terminal in this directory and run:
    ```bash
    npm install
    ```

2.  **Start Development Server**:
    To see the website in your browser, run:
    ```bash
    npm run dev
    ```
    Then open the URL shown in the terminal (usually `http://localhost:5173`).

3.  **Build for Production**:
    To create a production-ready build:
    ```bash
    npm run build
    ```

## Project Structure

- `public/frames/`: Contains the image sequence for the hero animation.
- `src/components/`: Reusable React components (Navbar, Hero, Services, etc.).
- `src/styles/`: Global styles and Tailwind configuration.
- `src/App.jsx`: Main application layout assembling all sections.

## Technologies

- **Vite**: Fast build tool and development server.
- **React**: UI library.
- **Tailwind CSS**: Utility-first CSS framework for styling.
- **Lucide React**: Icon set.
